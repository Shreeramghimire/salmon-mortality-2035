# salmon-mortality-2035
Open-source analytics project to support Norway's goal of 5% salmon mortality by 2035 using Python, R, SQL, Tableau, and AI

This open-source capstone project aims to support Norway's national goal of reducing salmon mortality to 5% by 2035. It uses statistical analysis, AI modeling, and interactive visualizations to identify key risk factors and track progress.

##  Objectives

- Identify key drivers of salmon mortality using historical data
- Develop statistical models to assess and forecast mortality
- Simulate scenarios and suggest data-driven interventions
- Track progress toward 5% mortality using time-series analysis
- Present findings via interactive dashboards and reports

##  Tools Used

- Python (pandas, scikit-learn, matplotlib)
- R (ggplot2, forecast, caret)
- SQL (PostgreSQL)
- Tableau Public
- Streamlit (optional)
- GitHub for version control and documentation
